Front-end Engineering Cheatsheet

Welcome to my cheatsheet. This is a WIP and will be updated as progress is made through the front-end engineering course.

CSS

Tag Name Description
Template Template Template
color Text Color Changes the text color
font-weight Font thickness Changes font thickness
background-color Background Color Changes background color of the element
font-style Font Styling Adds font style to the element
letter-spacing Letter Spacing Defines space between all the letters
*{} Resets Defaults Resets the sheet defaults
overflow Overflow Defines what the text does when too long. (hidden, scroll, visible)
display Display defines how elements sit together. Side by side or in order top to bottom. (inline, block, inline-block,none)
margin Margin Sets margin around the element. Set by top, right, bottom, left.
padding Padding Sets padding around the element. Set by top, right, bottom, left. The padding is inside the margin.
text-decoration Text Decoration Adds underline to the element.
:hover & :active Button actions when on hover and active hover has to come before active in the stylesheet.
Link styling correct order Correct order for the links/buttons to be formatted to work correctly :link :visited. :hover :active ; active is what happen when you click the button. Ex. it grows in size a tad
transition Transition property Specifies the time or color that the element will transition and how long that will take. Example a button expands size over .1s when hovered over.
